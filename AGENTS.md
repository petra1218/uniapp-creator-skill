## Agent skills

### Issue tracker

本仓库使用 GitHub Issues 作为 issue tracker。详见 `docs/agents/issue-tracker.md`。

### Triage labels

本仓库使用默认分诊标签：`needs-triage`、`needs-info`、`ready-for-agent`、`ready-for-human`、`wontfix`。详见 `docs/agents/triage-labels.md`。

### Domain docs

本仓库使用多上下文文档布局，由根目录 `CONTEXT-MAP.md` 指向多个上下文文档。详见 `docs/agents/domain.md`。
